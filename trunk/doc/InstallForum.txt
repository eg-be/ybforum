Installation
============
Den gesamten Inhalt aus 'web' an den entsprechenden
Ort auf dem Server kopieren.

Unter 'model/DbConfig.php' die Parameter für die Verbindung zur Datenbank
anpassen. Die Rechte der Datei anpassen (lesbar für www-data, sonst 
niemand).

In 'YbForumConfig.php' die benötigten Parameter anpassen:

 BASE_URL

 MAIL_FROM
 
Unter 'profile/ProfileConfig.php' das Profiling anpassen.

Den benötigten Benutzer für die Emails einrichten (pro-forma sollte 
die Adresse erreichbar sein.). Muss mit dem MAIL_FROM aus der 
Konfig übereinstimmen.

Einen Benutzer als Admin definieren, im Moment direkt in der DB:

UPDATE user_table SET admin = 1 WHERE nick = 'yb_wurst_macht_schön'

Anschliessend manuell die Liste der ausgeblendeten Posts prüfen und
Kinder auch ausblenden, wo notwendig.

Falls noch kein FULLTEXT Index für die Suche definiert ist, einen
solchen definieren:

ALTER TABLE post_table 
ADD FULLTEXT INDEX fulltext_title_content (title, content)
